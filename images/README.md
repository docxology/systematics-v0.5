## Adding New Images

1. Place image in this directory
2. Use descriptive filename (e.g., `feature_screenshot.png`)
3. Prefer PNG for screenshots, JPG for photographs
4. Keep file size reasonable (< 1MB recommended)
5. Update this README with the new image

## Usage in Markdown

```markdown
![Description](images/filename.png)
```

## Resolution Guidelines

| Type | Recommended |
|------|-------------|
| Screenshots | 2x resolution for retina |
| Diagrams | SVG preferred, PNG fallback |
| Photos | Compressed JPG |
